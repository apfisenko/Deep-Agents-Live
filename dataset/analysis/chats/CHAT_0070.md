# CHAT_0070 — разбор (уровень 1)

## Резюме
Долгий цикл: запрос **узкого продукта** (веб fullstack) → перенаправление на agents + B2B в разработке → отложенная покупка (занятость) → пожелание по контенту (фронт) → **реактивация** через 3+ месяца комбо-оффером 2026.

## Паттерны

| Паттерн | Триггер | Как формулируют | Исход |
|--------|---------|-----------------|-------|
| Запрос несуществующего/другого SKU | «курс по веб-проектам», «в записях» | Маппинг на agents + «корп. веб в разработке» | Частичное закрытие ожидания |
| Сроки потока | Загруженная неделя | «следующий поток», «набор в сентябре» | «Супер, поучаствую» |
| Product feedback | Уточнение программы | «больше внимания фронту», боль с компонентами | «Попробуем учесть» |
| Nurture / comeback | Пауза месяцы | Оффер 2026 с URL, «включена веб-разработка» | «Уже разобрался», совет по модулю веба |

## Сегментация
Смешение B2C запроса и упоминания **корпоративной** разработки — агенту нужно явно развести B2B/B2C.

## Частота
Запрос «другого курса, чем в каталоге» — **1/5**; отложенная покупка — **2/5**; реактивация — **2/5**.
