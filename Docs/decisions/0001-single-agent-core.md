# ADR-0001 — Один Agent Core, тонкие канальные адаптеры

> **Статус:** ✅ Accepted
> **Дата принятия:** 2026-06-06
> **Область:** Deep-Agents-Live

---

## Контекст

Нужны два канала — веб-виджет и Telegram — с идентичной бизнес-логикой (RAG, каталог, оплата, лиды). Дублирование логики в клиентах приведёт к расхождению поведения.

## Решение

Вся бизнес-логика сосредоточена в **Agent Core** (FastAPI + LangChain ReAct). Виджет (Next.js) и бот (aiogram в `frontend/bot/`) — тонкие адаптеры: передают сообщения в Core, получают ответ, форматируют под канал (`web` / `telegram`).

## Последствия

- Единая точка изменений для tools и RAG
- Бот и виджет деплоятся отдельно, но код бота — в репозитории `frontend/`
- Core обязан экспонировать API, пригодный для обоих каналов (в т.ч. SSE для web)
