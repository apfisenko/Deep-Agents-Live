# ADR-0002 — In-memory диалоги, без Postgres в MVP

> **Статус:** ✅ Accepted
> **Дата принятия:** 2026-06-06
> **Область:** Deep-Agents-Live

---

## Контекст

MVP — учебный стенд с фокусом на архитектуру агента, а не на персистентность. Postgres усложняет онбординг и не даёт критичной ценности на первом этапе.

## Решение

История диалогов хранится **in-memory** в Agent Core. При рестарте Core сессии сбрасываются. Лиды — в `data/leads.txt`. RAG — файлы в `data/b2b/`, `data/b2c/`. Postgres и `data-model.md` — вне MVP.

## Последствия

- Простой локальный запуск без миграций
- Нет восстановления диалога после рестарта — приемлемо для MVP
- Персистентность — в roadmap (Postgres)
